HTML Cheats

Boiler Plate

The boiler plate HTML code is there to setup the page ready to go. Its as follows:

<!DOCTYPE html>
<html>
  <header>
    <link href="stylesheet.css" type="text/css" rel="stylesheet">
    <title>Title of the page for the Tab</title>
  </header>
  <body>
  </body>
</html>

HTML Tags

Name Tag Use
Headings <h1>-<h6> Used to create headings of all sizes.
Paragraph <p> </p> Used to create paragraphs.
Linking Style Sheets <link href="" type="" rel=""> Used to link a CSS stylesheet to a HTML document.
Style Tag <style> </style> Used to create CSS styling within the html document
Span <span> </span> Used to add styling to a specific part of a tag rather than the whole thing.
Div <div> </div> Used to organise content for ease when adding CSS to the content as a whole.
Image <img src=""> Used to add an image.
Classes on a <p> Tag <p class=""> Used to add a class to a tag for more specific CSS use.
IDs on a <p> Tag <p id=""> Used to add an ID to a tag for the most specific option when using CSS.
Unordered List <ul> </ul> Used to create an unordered list.
Ordered List <ol> </ol> Used to create an ordered list.
List Item <li> </li> Used to create an item within a list.
Comment Tag <!– – …– –> Used to comment out text within html.
Table Setup <table> </table> Used to start a table.
Table Header <th> </th> Used to define the title headings along the top.
Table Cell <td> </td> Used to create the cells of the table.
Table Row <tr> </tr> Used to create a new row within the table.
Audio <audio>
  <source src="file.mp3" type="audio.mpeg"/>
  Your browser does not support this audio file.
</audio>		 Used to add an audio file to the page as well as a message if the file cannot load.
Audio Autoplay <audio autoplay> </audio> Used to play the audio file as soon as it's ready.
Audio Controls <audio controls> </audio> Used to add play and pause buttons etc to the audio file on screen.
Audio Loop <audio loop> </audio> Used to make the audio file play on a loop.
Audio Muted <audio muted> </audio> Used to mute the audio file.
Audio Source <audio src> </audio> Used to specify the URL of the audio file.
Audio Preload <audio preload="auto"> </audio> Used to let the author decide when the audio file loads.
"auto" loads entire file upon page load.
"metadata" loads file when loading metadata.
"none" does'nt load when page loads.

HTML Semantic Elements

Name Tag Use
Article <article> </article> Used to specify independant content e.g. makes sense on its own.
Aside <aside> </aside> Content such as a sidebar or something that relates to but is not 100% necessary to the main content.
Details <details> </details> Used to define additional content that the user can view or hide.
Figcaption <figcaption> </figcaption> Used to add a caption the the "Figure" content as to offer a worded explanation of the content.
Figure <figure> </figure> Used to tag illustrations, picturs, diagrams etc.
Footer <footer> </footer> Used to define the footer of a doucment or section e.g. copyright, authorship or contact info.
Links to the top of the page, related docs or a sitemap.
Header <header> </header> Used to hold things such as navigation bars, logos or icons.
Main <main> </main> Used to specify the main section of the content of the whole document.
Mark <mark> </mark> Used to define marked or highlighted text.
Nav <nav> </nav> Used for a navigation bar or a group of links and not just any hyperlink.
Section <section> </section> Used to split the content into sections e.g. Intro, Main content, Contact Info.
Summary <summary> </summary> Used to define a visbile heading that contains a "Details" tag.
Time <time> </time> Defines a date and time.

CSS Cheats

CSS Tags

Name Tag Use
Regular Tags (Generetic) h1 { Used to target a common simple tag in the HTML.
Classes (More specific) .class-name { Used to target a specific tag that has the class name assigned to it.
IDs (Most specific) #id-name { Used to more specifically target a tag, perhaps already has a class tag on both.
Multiple Tags h1, h2, p { A comma is used to target multiple tags with the same styling.
Important Ability .class-name {
  color: red !important;		 Used to override any other styling and use this one.
Specifics in Tables table td:first-child { Used to target the first <td> of a section.
last-child or nth-child(2) can also be used to target different tags.
Padding p {
  padding: 2em, 1em;		 Used to control the padding of the content from the edge of the tag itself.
Tag order is; Top, Right, Bottom, Left. Can be simplified if padding is similar for each side.
Margin p {
  margin: auto;		 Used to control the margin of the content on the page from other tags.
Tag order is; Top, Right, Bottom, Left. Can be simplified if padding is similar for each side.
No Wrapping Text p {
  white-space: nowrap;		 Used to ensure that the text doesn't wrap when screen size is changed.
Borders p {
  border-style: solid;
  border-radius: 30px;		 Used to add a border to a tag with a certain style
